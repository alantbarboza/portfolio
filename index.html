Projetos

Gerador de Currículo PDF

Este projeto é um gerador de currículos no modelo ATS, permitindo que os usuários criem, editem e exportem seus currículos em formato PDF, tudo no client-side (lado do cliente) diretamente no navegador.

Funcionalidades:
Edição direta no navegador: Os campos são editáveis diretamente na página.

Salvar e carregar dados: Possibilidade de salvar as informações no navegador (local storage), salvar um backup.txt e carregar posteriormente.

Exportar para PDF: Geração de um PDF do currículo com texto selecionável e quebra de página ("page break" / "autopaging") para evitar cortes no meio do conteúdo.

Importar backup: Permite importar arquivos.txt de backup.

Adicionar e remover seções: Possibilidade de adicionar novas experiências, habilidades e outras informações.

Extensão Raposa

Extensão para navegador que facilita tarefas repetitivas e manipulação de texto.

Funcionalidades:
Concatenar linhas: Adiciona aspas simples no início e fim de cada linha.

Copiar texto: Permite copiar o texto concatenado para a área de transferência.

Bloco de notas: Área para anotações rápidas.

Macros: Automatiza ações como mudar de aba ou atualizar a página.

A.Souza Elétrica

Website desenvolvido para a empresa A.Souza Elétrica.

Funcionalidades:
Página inicial: Apresentação da empresa.

Sobre: Lista detalhada dos serviços oferecidos.

Projetos: Exibe imagens dos trabalhos realizados, integrando diretamente o feed do Instagram.

Formulário de contato: Permite envio de mensagens diretamente pelo site.

Design responsivo: Compatível com dispositivos móveis e desktops.

Processamento de Dados CSV

Script em Python que automatiza o processamento de dados financeiros em arquivos CSV.

Funcionalidades:
Filtragem inteligente: Identifica automaticamente transações acima de R$1000.

Registro de logs: Erros e informações do processamento são registrados em um arquivo de log.

Saída no terminal: Mostra o resultado do processamento diretamente no console.

Exportação de resultados: Gera novos arquivos CSV com os dados filtrados.

Dados do Clima

Projeto em Python que realiza a integração com a API OpenWeatherMap para consultar informações meteorológicas em tempo real de qualquer cidade.

Funcionalidades:
Conexão com a API: Realiza solicitações HTTP utilizando o método GET para obter os dados do clima.

Autenticação via chave de API: Garante acesso autorizado à API pública do OpenWeatherMap.

Consulta personalizada: Permite definir a cidade de interesse e retornar dados em tempo real.

Informações exibidas: Descrição detalhada do clima (ex: ensolarado, chuvoso) e temperatura atual em graus Celsius.

YouTube Downloader (Áudio e Vídeo)

Script em Python que utiliza a biblioteca yt-dlp para baixar vídeos ou áudios do YouTube com exibição de progresso e escolha de qualidade.

Funcionalidades:
Suporte a múltiplos formatos: Permite baixar apenas o áudio (m4a, best) ou o vídeo (mp4).

Seleção de qualidade: Opção de escolher entre diferentes resoluções disponíveis (720p, 1080p, etc).

Exibição de progresso: Mostra o andamento do download, velocidade e tempo restante diretamente no terminal.

Evita bloqueios regionais: Utiliza cabeçalhos HTTP simulando um navegador para garantir estabilidade e compatibilidade com o YouTube.

Gerenciador de Tarefas

Aplicativo desktop desenvolvido com Electron, voltado para organização pessoal e gerenciamento de tarefas.

Funcionalidades:
Criar, editar e excluir tarefas: Gestão completa das atividades.

Armazenamento local: Salva os dados no computador do usuário.

Interface intuitiva: Facilita a visualização e manipulação das tarefas.

API To-Do List (Autenticação e Autorização)

API desenvolvida em Node.js com Prisma ORM para gerenciamento de tarefas, implementando autenticação e autorização.

Funcionalidades:
Autenticação: Registro e login de usuários com geração de tokens JWT.

Autorização baseada em papéis: Controle de acesso a rotas com diferentes níveis de permissão.

CRUD completo: Operações de criação, leitura, atualização e exclusão de tarefas vinculadas a cada usuário.

Integração com banco de dados: Utilização do Prisma ORM com suporte a MySQL.

Estrutura do projeto:
controllers, middlewares, models, repositories, routes, schemas, services, utils e server.

Chat Fake

Chat Fake é uma simulação de um aplicativo de mensagens desenvolvido com HTML5, CSS3 e JavaScript puro. Oferece uma experiência interativa e realista de um aplicativo de chat.

Funcionalidades:
Lista de contatos: Acesse todos os contatos disponíveis.

Busca de contatos: Filtro rápido para encontrar quem você procura.

Histórico de mensagens: Veja todas as conversas anteriores com os contatos.

Busca em mensagens: Encontre mensagens específicas usando a barra de busca integrada.

Reações com emojis: Reaja às mensagens com emojis disponíveis no aplicativo.

Notificações: O título da aba pisca para alertar sobre novas mensagens quando o usuário não está com a aba ativa.

Resposta automática: O bot responde suas mensagens após alguns segundos de forma automática.

Contatos verificados: Alguns contatos possuem badge de verificação para identificação especial.
